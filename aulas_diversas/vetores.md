# Usando laços de repetição em vetores
* crtl + shift + l alterar valores iguais
### Como fizemos:

```python 
numeros = [10,5,6,7,8]
print("1",numeros[0])
print("2",numeros[1])
print("3",numeros[2])
print("4",numeros[3])
print("5",numeros[4])
```

### Com laço de repetição:
```python
for i in range(len(numeros)):
    print(numeros[i])
```
* também posso fazer isso:
```python
for numero in numeros:
     print(numero)
```
